## Getting started

1. Clone this repo

```bash
git clone https://github.com/dabit3/react-native-navigation-v2.git
```

2. Change into the directory

```bash
cd react-native-navigation-v2
```

3 Install the dependencies

```bash
yarn
# or
npm install
```

4. Run the project

```bash
react-native run-ios
# or
react-native run-android
```
